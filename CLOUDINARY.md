# Інструкція Cloudinary для фотографій сайту

Ця інструкція пояснює, як завантажити фотографії в Cloudinary та додати їх на сайт ліцею.

## 1. Створіть або відкрийте акаунт Cloudinary

1. Перейдіть на сайт Cloudinary: `https://cloudinary.com`.
2. Увійдіть у свій акаунт або створіть новий.
3. Відкрийте розділ `Media Library`.

## 2. Завантажте фотографії

1. У `Media Library` натисніть `Upload`.
2. Виберіть фотографії з комп'ютера.
3. Дочекайтеся завершення завантаження.
4. Для зручності можна створити папки, наприклад:
   - `news`
   - `events`
   - `lessons`
   - `school-life`

## 3. Скопіюйте посилання на фото

1. Натисніть на завантажене фото в `Media Library`.
2. Знайдіть поле з URL або кнопку копіювання посилання.
3. Скопіюйте пряме посилання на зображення.

Зазвичай посилання має такий вигляд:

```text
https://res.cloudinary.com/your-cloud-name/image/upload/v1234567890/folder/photo.jpg
```

Для сайту потрібні саме такі публічні HTTPS-посилання, які починаються з:

```text
https://res.cloudinary.com/
```

## 4. Додайте фото через адмінку сайту

1. Відкрийте адмінку:

```text
admin/index.html
```

або локально:

```text
http://127.0.0.1:8000/admin/index.html
```

2. Введіть пароль.
3. У блоці `Додати фотографії` виберіть розділ.
4. Заповніть `Назва блоку`.
5. У поле `Посилання Cloudinary` вставте посилання на фото.
6. Кожне посилання вставляйте з нового рядка.
7. Натисніть `Додати`.
8. У блоці `Збереження` натисніть `photos.json`.
9. Замініть завантаженим файлом файл:

```text
data/photos.json
```

10. Зробіть commit і push на GitHub.

## 5. Додайте фото вручну в `photos.json`

Можна редагувати файл напряму:

```json
[
  {
    "section": "Заходи",
    "title": "День знань",
    "images": [
      "https://res.cloudinary.com/your-cloud-name/image/upload/v1234567890/events/photo-1.jpg",
      "https://res.cloudinary.com/your-cloud-name/image/upload/v1234567890/events/photo-2.jpg"
    ]
  }
]
```

Доступні розділи для фотографій:

- `Заходи`
- `Уроки`
- `Шкільне життя`

## 6. Оптимізація фотографій

Cloudinary дозволяє змінювати розмір і якість фото прямо через URL.

Наприклад, оригінальне посилання:

```text
https://res.cloudinary.com/your-cloud-name/image/upload/v1234567890/events/photo.jpg
```

Оптимізоване посилання:

```text
https://res.cloudinary.com/your-cloud-name/image/upload/f_auto,q_auto,w_1200/v1234567890/events/photo.jpg
```

Що означають параметри:

- `f_auto` - автоматично вибрати найкращий формат для браузера.
- `q_auto` - автоматично підібрати якість і зменшити розмір файлу.
- `w_1200` - обмежити ширину зображення до 1200 px.

Для цього сайту можна використовувати оптимізовані посилання в `data/photos.json`.

## 7. Публікація змін на GitHub

Після заміни `data/photos.json` виконайте:

```bash
git add data/photos.json
git commit -m "update photos"
git push
```

Після цього GitHub Pages оновить сайт. Іноді оновлення займає кілька хвилин.

## Важливо

- Не вставляйте посилання з особистого кабінету Cloudinary, які працюють лише після входу.
- Використовуйте публічні HTTPS-посилання з домену `res.cloudinary.com`.
- Не видаляйте фото з Cloudinary, якщо вони вже використовуються на сайті.
- Якщо фото не відкривається в окремій вкладці браузера, воно не відобразиться й на сайті.
